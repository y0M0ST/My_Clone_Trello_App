# 🚀 Trello Clone - Task Management App

Một ứng dụng quản lý công việc theo phong cách Kanban (giống Trello), hỗ trợ kéo thả mượt mà, quản lý Board, List, Card và làm việc nhóm hiệu quả.

## 🛠️ Công nghệ sử dụng (Tech Stack)

**Frontend:**
- ⚛️ React.js + TypeScript
- 🎨 Tailwind CSS + Shadcn UI
- ✋ @dnd-kit (Drag & Drop)
- 📡 Axios + React Query

**Backend:**
- 🟢 Node.js + Express
- 🗄️ Database : PostgreSQL
- ☁️ Cloudinary (Upload ảnh)
- 🔒 JWT Authentication
